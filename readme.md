# To Do List 프로젝트
https://oz528.github.io/momentum/

오늘의 할 일 리스트, 현재 내 위치의 간단한 정보를 받아와서 표시해주는 프로젝트입니다.

## 구현내용
- Javascript를 이용해 현재 위치값을 받아오고, 실시간 시계를 표시했습니다.
- Local Storage를 이용해 개인화 설정을 유지시켰습니다.
- 기본에 충실하기 위해서 scss가 아닌 css를 사용했습니다.
- 반응형 페이지로 만들었습니다.
